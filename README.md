# pick - A simple library for picking random elements of arrays

# REQUIREMENTS

 - [Node.js](http://nodejs.org/)

# DEVELOPMENT

# Linting

Keep the code tidy:

    $ npm run-script lint

# EXAMPLE

    $ git clone https://github.com/mcandre/node-pick.git
    $ cd node-pick
    $ node
    > var pick = require("pick");
    > pick.pick([1, 2, 3]);
    1
