# pick - A simple library for picking random elements of arrays

# EXAMPLE

    $ node
    > var pick = require("pick");
    > pick.pick([1, 2, 3]);
    1

# HOMEPAGE

https://github.com/mcandre/node-pick

# REQUIREMENTS

* [Node.js](http://nodejs.org/)

# INSTALL

    $ npm install pick

# LINT

Keep the code tidy:

    $ npm run-script lint
